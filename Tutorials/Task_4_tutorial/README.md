# Task 4 Starter Kit: AI for Venture Capital

This starter kit provides a template for the **AI for Venture Capital** task.


## 📂 Directory Structure

```
Task_4_tutorial/
├── README.md               # This file
├── Task_4_Description.md   # Task description and dataset info
├── requirements.txt        # Python dependencies
├── start.bash              # Start main.py script
├── data_loader.py          # Script to load VCBench
├── example_model.py        # Template model
├── evaluate.py             # F-score Evaluation
└── main.py                 # Main script to run inference
```

## 🚀 Setup

```bash
cd Tutorials/Task_4_tutorial
pip install -r requirements.txt
```

## ▶️ Running

```bash
bash start.bash
```

## 🛠 Developing Your Model

This example provide a pure method to use LLM predict the VCBench. 
The most important part is the output file format of the VCbench_dev.csv.
You can develop your own method but please follow the example code to save the output for submission.
